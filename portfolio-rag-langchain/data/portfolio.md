# How to use this portfolio site

On the Skills and Projects sections of this portfolio, each card is hoverable - hovering over
a project or skill card reveals additional details, including the stats bars described below.

The stats/progress bars shown on both the Skills cards and the Project cards are not the same
thing:

- On a **Project card**, the stats show the specific tools and technologies used on that
  particular project (for example, "React.js", "Tailwind CSS", "EmailJS"). They indicate what
  was used to build that project, not a proficiency rating.
- On a **Skills card**, the stats show how knowledgeable Carl is in each language, framework,
  or tool listed under that category (Programming Languages, Frontend, Backend, Tools). The
  percentage reflects his self-assessed familiarity/proficiency with that specific skill.

# Projects

## TRipplesDigital Website (2026)

During his internship at TRipplesPH, Carl was tasked with developing a static corporate
website for TRipples Digital. He built the website using React and Tailwind CSS, integrated
EmailJS for contact form submissions, and handled the deployment to a production environment
through cPanel.

Tools used: React.js, Tailwind CSS, EmailJS, cPanel.

## WFH Attendance (2026)

During his internship at TRipplesPH, Carl identified challenges in tracking and reviewing
work-from-home attendance records. To address this, he proposed and developed an automated
attendance system using Telegram, n8n, and Google Sheets. Employees can record their time-in
and time-out through a Telegram bot, and the data is automatically logged into Google Sheets
for easier monitoring and reporting. The solution was adopted and used by the company.

Tools used: n8n, Telegram Bot, Google Sheets, Automation.

## LMS - EduLink (2025)

EduLink is a Learning Management System (LMS) capstone web application. As part of a group
project, Carl served as the programmer, responsible for coding and deployment.

Tools used: Next.ts, Prisma ORM, PostgreSQL, Docker.

## Inventory System (2025)

This was a freelance project for one of Carl's subjects. His client was Jep Shop, a motorcycle

Tools used: MongoDB, Express.js, React.js, Node.js.
